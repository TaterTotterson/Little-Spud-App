# Little Spud App Review Notes

Little Spud is a companion app for Tater, a self-hosted local-first assistant. Normal use requires pairing with a user's private Tater instance by scanning a pairing QR code. There is no public hosted account system.

For App Review, please use the built-in offline demo mode:

1. Open Little Spud.
2. Tap `Try Demo` on the pairing screen.
3. Send a message such as `Show me a sample image`.
4. Turn on TTS with the speaker button to hear local speech playback.
5. Tap the bell button to test local iOS notifications.
6. Tap the microphone button to test the voice input flow.

The demo mode runs locally on the device and does not connect to any private Tater server. It is included so App Review can verify the app UI, chat flow, tool-progress messages, media display, local notifications, TTS, and voice controls without access to a user's private home system.

Users with their own Tater server can leave demo mode through `Forget Pairing`, then scan a pairing QR code from their Tater WebUI.
